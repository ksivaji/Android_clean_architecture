# Github User App


A sample android app showcasing MVVM architecture using Android
Architecture components.

![](screenshots/guide-to-app-architecture.png )


# **Libraries used**


* [Android KTX][2]
 * [Data Binding][11]
  * [Lifecycles][12]
  * [LiveData][13]
  * [Room][16]
  * [ViewModel][17]
  * [Paging][19]
  * [Kotlin Coroutines][91]
  * [Dagger-android][92]
  * [Retrofit 2][93]
  * [OkHttp 3][94]
  * [GSON][95]
  * [Glide][90]
  * [Timber][96]
  * [Stetho][97]
